Informações

Tipos

  • Number - Números Inteiros, Decimais e Negativos
  • String - Cadeia de Caracteres
  • Bool - True ou False
  • Null - Vazio
  • Undefined - Não Definido

Operações

  • [ + ] - Soma
  • [ - ] - Subtração
  • [ * ] - Multiplicação
  • [ / ] - Divisão
  • [ % ] - Resto da Divisão

Strings

  • "string" + "string" - Concatenação das Strings
  • "string" [0] - Busca por Index
  • "string".length - Exibe o Tamanho da String

Interação

  • console.log("JavaScript Developer") - Imprime no Console
  • console.info("JavaScript Developer") - Exibe uma Informação
  • console.warn("JavaScript Developer") - Exibe um Alerta
  • console.error("JavaScript Developer") - Exibe um Erro
  • alert("Olá Dev!") - Exibe um Alerta
  • confirm("Deseja se tornar um Dev Full Stack?") - Exibe uma Confirmação
  • prompt("Qual o seu nome?") - Exibe um Inserção de Dados

Variáveis

    1. LET - INFORMA QUE ESTÁ DECLARANDO UMA VARIÁVEL

        2. let spaceship = "ApolloX 2030"

        console.log(spaceship)

        spaceship = "Dragon Kondraz"

        console.log(spaceship)

    2. CONST - UMA VARIÁVEL RECEBE UM VALOR CONSTANTE

        spaceship = "ApolloX 2030"

        console.log(spaceship)

        spaceship = "Dragon Kondraz"

        console.log(spaceship)

    3. VAR - MESMO QUE O LET

        var spaceship = "ApolloX 2030"

        console.log(spaceship)

    4. UNDEFINED - NÃO INFORMA NENHUM VALOR

        let no_value

        console.log(no_value)

    5. UNDEFINED - NÃO INFORMA NENHUM VALOR

        let no_value

        console.log(no_value)
